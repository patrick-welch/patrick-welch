## Selected Work

### [soft-mmu-68k](https://github.com/patrick-welch/soft-mmu-68k)

A work-in-progress soft memory-management unit for Motorola 68K-class systems, inspired by the MC68851/MC68030 PMMU model.

This project is my main architecture and verification lab. It explores address translation, permission checks, function-code behavior, TLB-style structures, flush/test operations, simulation-driven validation, and eventual FPGA implementation.

The goal is not nostalgia for its own sake. The goal is to use a historically important architecture as a concrete testbed for modern HDL, verification, documentation, and AI-assisted development workflows.
